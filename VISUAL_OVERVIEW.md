## Key Visual Elements

### Color Coding
- **Status Badges**: 
  - Lead: Purple gradient
  - Quoted: Pink gradient  
  - In Progress: Blue gradient
  - On Hold: Red gradient
  - Completed: Green gradient
  - Cancelled: Gray gradient

- **Category Badges**:
  - Each brand category has a unique gradient color
  - Required brands shown with green checkmark
  - Action items: Green (completed), Gray (pending)

### Interaction Patterns
- **Hover Effects**: Cards lift up with shadow on hover
- **Click Animations**: Buttons scale down slightly on click
- **Modal Animations**: Slide and fade in/out smoothly
- **Search**: Real-time filtering as you type
- **Forms**: Multi-step with add/remove dynamic fields

### Responsive Design
- **Desktop**: Full sidebar + content area
- **Tablet**: Collapsible sidebar
- **Mobile**: Hamburger menu, stacked cards
