# Quick Start Guide - Using the CRM

## Accessing the CRM

1. Navigate to your CRM URL (e.g., https://your-app.vercel.app)
2. Login with your credentials
3. You'll see the Dashboard with different tabs

## Overview Tab

The Overview tab shows you a quick summary:
- **Welcome Banner**: Personalized greeting
- **Statistics Cards**: Key metrics at a glance
  - Total Meetings this month
  - Active Projects
  - Total Clients
  - Brands Managed
- **Recent Activity**: Latest actions in the system

## Managing Meetings

### Creating a New Meeting
1. Click on the **Meetings** tab
2. Click the **+ New Meeting** button (top right)
3. Fill in the meeting details:
   - Select the **Client** (required)
   - Optionally select a **Project**
   - Set the **Meeting Date**
   - Enter the **Location**
   - Add a **Summary** of the meeting
4. Add **Brands Discussed**:
   - Click **+ Add Brand**
   - Select the brand from dropdown
   - Check "Product Required" if the client needs this brand
   - Add estimated value if discussed
   - Add notes about the discussion
   - Repeat for multiple brands
5. Add **Action Items**:
   - Click **+ Add Action Item**
   - Describe the task
   - Set a due date
   - Repeat for multiple action items
6. Click **Create Meeting**

### Editing a Meeting
1. Click the **Edit** button (pencil icon) on any meeting card
2. Make your changes
3. Click **Update Meeting**

### Deleting a Meeting
1. Click the **Delete** button (trash icon) on the meeting card
2. Confirm the deletion

### Searching Meetings
- Use the search box at the top to find meetings by client name, company, or location

## Managing Projects

### Creating a New Project
1. Click on the **Projects** tab
2. Click the **+ New Project** button
3. Fill in the project details:
   - **Project Name** (required)
   - **Project Number** (optional, for reference)
   - Select **Client** (required)
   - Choose **Status** (Lead, Quoted, In Progress, etc.)
   - Enter **Location**
   - Add **Estimated Value** in Rands
   - Set **Start Date** and **End Date**
   - Add a **Description**
4. Select **Brands Involved**:
   - Check the boxes for all brands used in this project
5. Click **Create Project**

### Filtering Projects
- Use the **status filter buttons** (All, Lead, Quoted, etc.) to view specific types
- Use the **search box** to find projects by name, number, or location

### Project Status Meanings
- **Lead**: Potential project, initial contact made
- **Quoted**: Proposal/quote sent to client
- **In Progress**: Active construction/implementation
- **On Hold**: Temporarily paused
- **Completed**: Finished and delivered
- **Cancelled**: Not proceeding

## Managing Brands

### Adding a New Brand
1. Click on the **Brands** tab
2. Click **+ New Brand** (admin/staff only)
3. Enter brand information:
   - **Brand Name** (required)
   - **Category** (required) - e.g., Windows & Doors, Security
   - **Description** - product line details
   - **Contact Name** - your contact at the brand
   - **Contact Email**
   - **Contact Phone**
   - **Website** - will become a clickable link
   - **Notes** - any additional information
4. Click **Create Brand**

### Filtering Brands
- Use the **category filter buttons** to view specific types
- Use **search** to find brands by name or contact

## Generating Reports

### Creating a Report
1. Click on the **Reports** tab
2. Set your **date range**:
   - Choose **Start Date**
   - Choose **End Date**
3. Optionally filter by a specific **Brand**
4. View the statistics and tables

### Understanding Report Metrics

**Summary Statistics**:
- **Total Meetings**: Number of meetings in the period
- **Unique Clients**: How many different clients you met with
- **Brands Discussed**: Number of unique brands mentioned
- **Hit Rate**: Percentage of brand discussions where product was required
- **Discussion Value**: Total estimated value from all brand discussions
- **Active Projects**: Projects started in this period
- **Project Value**: Combined value of projects in period

**Brand Performance Table**:
- **Discussions**: How many times brand was mentioned
- **Required**: How many times product was actually needed
- **Hit Rate**: Individual brand success rate
- **Est. Value**: Total potential value for this brand

**Client Activity Table**:
- **Meetings**: How many meetings per client
- **Brands Discussed**: Variety of brands discussed with each client

### Exporting Reports
1. Review your report data
2. Click **Export CSV** button (top right)
3. The CSV file will download with all data formatted for Excel/Google Sheets

## Managing Your Settings

### Updating Your Profile
1. Click on the **Settings** tab
2. Select **Profile** (left sidebar)
3. Update your:
   - **Full Name**
   - **Location** (JHB, Cape Town, Durban, Other)
4. Click **Save Changes**

Note: Email and Role cannot be changed - contact your administrator

### Changing Your Password
1. In Settings, select **Password** (left sidebar)
2. Enter your **New Password** (min 8 characters)
3. **Confirm** your new password
4. Click **Change Password**

### Setting Notification Preferences
1. In Settings, select **Notifications** (left sidebar)
2. Toggle preferences:
   - **Email Notifications** - general updates
   - **Meeting Reminders** - before meetings
   - **Project Updates** - when projects change
   - **Monthly Reports** - automated monthly summaries
3. Click **Save Preferences**

## Role-Based Features

### Admin & Staff
- Can create, edit, and delete all records
- Full access to all features
- Can manage brands

### Brand Representatives
- Can view meetings mentioning their brand(s)
- Can create meetings
- Can view projects with their brand(s)
- Can generate reports

### Contractors
- **Read-only access**
- Can view their meetings
- Can view projects they're involved with
- Cannot create or edit records

### Suppliers
- **Read-only access**
- Can view meetings discussing their brand(s)
- Can view projects using their brand(s)
- Cannot create or edit records

## Tips for Best Results

### Meetings
- ✅ Record meetings as soon as possible while details are fresh
- ✅ Always add brand discussions to track performance
- ✅ Create action items for follow-ups
- ✅ Link meetings to projects when relevant

### Projects
- ✅ Update status regularly to keep team informed
- ✅ Add all brands being specified
- ✅ Include estimated values for accurate reporting
- ✅ Keep project numbers consistent with your system

### Brands
- ✅ Keep contact information up to date
- ✅ Use consistent categories for easy filtering
- ✅ Add website links for quick reference

### Reports
- ✅ Generate monthly reports for performance tracking
- ✅ Use brand filtering to analyze individual brand success
- ✅ Export to CSV for presentations to management
- ✅ Track hit rates to identify successful brands

## Getting Help

If you encounter issues:
1. Check that you're using a supported browser (Chrome, Firefox, Safari, Edge)
2. Try refreshing the page
3. Check your internet connection
4. Clear browser cache if forms aren't loading
5. Contact your system administrator

## Keyboard Shortcuts

- **Tab**: Move between form fields
- **Enter**: Submit forms (in most cases)
- **Esc**: Close modal windows
- **Ctrl/Cmd + F**: Browser search (find on page)
